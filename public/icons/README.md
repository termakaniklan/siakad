# PWA Icons

Replace the placeholder icons in this directory before deploying to production. The PWA
manifest references the following sizes:

- `icon-192.png` — 192×192, used for app shortcuts and Android home screen.
- `icon-512.png` — 512×512, used for splash screen and high-DPI installs.
- `icon-maskable-512.png` — 512×512, masked safe-zone variant for Android adaptive icons.

The bundled placeholders are minimal solid-color PNGs so the manifest validates and the
app installs out of the box. Overwrite them with the school logo and re-deploy.
